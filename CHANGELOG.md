## [0.5.0] - 2026-01-03

### Added

- Built-in OmniObserve `llmops` provider adapter
- `llmops/provider.go` - Full `llmops.Provider` implementation
- `llmops/trace.go` - Trace and span adapters
- `GetDatasetByID` for retrieving datasets by ID
- `DeleteDataset` for removing datasets
- `CreateAnnotation` and `ListAnnotations` stubs (returns not implemented)
- Integration tests for SDK and llmops provider
- Feature comparison matrix in README

### Changed

- **BREAKING:** Module renamed from `github.com/agentplexus/go-comet-ml-opik` to `github.com/agentplexus/go-opik`
- Updated dependency: `omniobserve v0.4.0` to `v0.5.0`

## [0.1.0] - 2025-12-20

### Added

- Initial release of go-opik SDK
- Core SDK: client, traces, spans, context propagation
- Data management: datasets, experiments, prompts
- Evaluation framework: heuristic metrics, LLM judges
- Integrations: OpenAI, Anthropic tracing transports
- CLI tool (`cmd/opik`)
- HTTP middleware for automatic tracing
- Streaming and batching support
- MkDocs documentation site
